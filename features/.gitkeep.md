# Features Directory

This directory contains feature specifications, plans, and tasks.

Features are organized as:
- `NNN-feature-name.spec.md` - Feature specification
- `NNN-feature-name.plan.md` - Technical implementation plan
- `NNN-feature-name.tasks.md` - Implementation tasks checklist
- `NNN-feature-name/status.md` - Task tracking and progress

See `/skills/README.md` for the feature development workflow.
